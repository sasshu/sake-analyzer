※IDとパスワードによるログインが必要です。

ID：cooperator

パスワード：sake_0117_key

## アプリ概要
日本酒の成分や製法のデータを可視化するWebアプリケーションです。

絞り込み機能により、様々な条件を指定してデータをグラフに出力することができます。
